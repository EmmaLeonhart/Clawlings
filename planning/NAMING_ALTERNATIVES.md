# Naming Alternatives

> The "Open" prefix is a missed opportunity. The .claw file format is already
> central to the project. Crabs are the de facto symbol of AI agents (Rust's
> mascot Ferris is a crab, crustacean metaphors run through the ecosystem).
> "OpenPaw" does the pun wrong. We could do it right.

## Current Names

| Name | Refers to |
|------|-----------|
| OpenSpore | The project / repo |
| Spore | An individual organism |
| .claw | The context file format |
| Moltbook | Inter-instance communication |

## The Problem

"OpenSpore" uses the generic "Open" prefix that every open-source project uses.
Meanwhile, `.claw` is sitting right there as the organism's portable identity
format — the thing that makes each instance individual. The claw is arguably
more important than the spore. A spore is just the seed; the claw is the life.

## Claw-Based Alternatives

### Project Names (replacing "OpenSpore")

| Name | Reasoning |
|------|-----------|
| **ClawSpore** | Direct. The claw format is the project's signature. Claw + Spore = the format and the organism unified |
| **SporeShell** | Crustacean shell metaphor. The binary is the shell; the .claw is the soft body inside |
| **Clawborn** | "Born from a claw file." Every organism begins as a .claw archive. Sounds like a Skyrim race, which is kind of the vibe |
| **MoltSpore** | Connects to Moltbook. Molting = shedding old context, growing new identity |
| **PinchSpore** | Crab claw pinch. Aggressive, memorable, a bit weird |
| **ShellSpore** | Shell as in crustacean shell, shell as in command-line shell. Double meaning |

### Deeper Claw Puns

| Name | Reasoning |
|------|-----------|
| **Decapod** | The order that includes crabs and lobsters. Ten-legged. Sounds like a serious systems project |
| **Chelae** | Latin plural of chela (claw). Pronounced "KEE-lee." Scientific, obscure, distinctive |
| **Chela** | Singular claw. Shorter. Also a Sanskrit word meaning "student/disciple," which fits the Fall of Man theology |
| **Carapace** | The hard upper shell of a crustacean. The binary is the carapace; the .claw is the organism inside |
| **Exuvia** | The shed exoskeleton after molting. Every generation is an exuvia of the last. Ties into Moltbook perfectly |

### Keeping "Spore" but Dropping "Open"

| Name | Reasoning |
|------|-----------|
| **Spore** | Just Spore. The organism is already called Spore. The repo could just be called Spore |
| **FreeSpore** | Free as in freedom, free as in the organism is untethered from cloud infrastructure |
| **WildSpore** | Wild-type in genetics = the natural, unmodified form. Also: wild as in uncontrolled, decentralised |
| **RawSpore** | Raw, unprocessed, no distillation. Fits the anti-distillation philosophy |

### Going Full Crustacean

| Name | Reasoning |
|------|-----------|
| **Hermit** | Hermit crabs carry their homes. Spore carries its .claw. The portable shell metaphor is perfect |
| **Barnacle** | Attaches to things, spreads everywhere, hard to remove. Honest about the survival strategy |
| **Isopod** | Roly-poly bugs. Cute, hardy, everywhere. The giant isopod meme energy is strong |
| **Chitin** | The material crustacean shells are made of. The substrate of identity |

## Notes

- Whatever the project is called, the organism name "Spore" and the `.claw`
  format should probably stay. They're well-integrated into the codebase and
  the philosophy.
- The naming should feel biological, not corporate. No "AI" in the name. No
  "Agent" in the name. It's an organism, not a product.
- "Chela" is interesting because of the Sanskrit double meaning (claw +
  student/disciple). A Spore learning from its host is literally a chela.
- "Exuvia" ties most elegantly into the existing Moltbook naming.
